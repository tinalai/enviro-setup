### Terminal Alias:
```# ----------------------
# General Aliases
# ----------------------

#---- ls command output ----#

## Colorize ls output ##
alias ls='ls --color=auto'

## Long listing format ##
alias ll='ls -la'

## Show hidden files ##
alias l.='ls -d .* --color=auto'

#---- cd command behaviour ----#

## Get rid of command not found ##
alias cd..='cd ..'

## Shortcut to get out of cuurent directory ##
alias ..='cd ..'
alias ...='cd ../../../'
alias ....='cd ../../../../'
alias .....='cd ../../../../../'
alias .4='cd ../../../../'
alias .5='cd ../../../../../'
```
